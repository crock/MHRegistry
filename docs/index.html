MH Registry

Minecraft Server Plugin

Build Registry Plugin for Minecraft Servers

MHRegistry is a simple Minecraft server plugin to read and count the various block types inside a WorldGuard region and send that data to a configurable rest api endpoint.

It can be used to enrich web directories with valuable data. For instance, on a directory of Minecraft builds, you should show the total number and types of Minecraft blocks that make up any build in the directory.

The source code is available to view on GitHub. I am currently accepting contributions from both new and experienced developers. I actively tag issues with good-first-issue to help new developers get valuable experience to land them a job in tech.

Plugin Usage

Make sure both WorldGuard and WorldEdit are installed or the plugin won't work.

First, you need to define a new WorldGuard region.

Then, you can run the command /submitbuild.

/submitbuild <build name> [region name]

Permissions

The permission node for the command is mhregistry.submit and defaults to OP. This allows a user to submit builds to the configured api endpoint.
